# 🎓 Student Attendance Management System

🚀 A web-based **Student Attendance Management System** built using **Python & Django**.  
This project helps educational institutes efficiently manage student attendance, records, and reports through a centralized system.

---

## 🌍 Project Overview

This system allows admins and teachers to manage student data, mark attendance, and generate reports easily.  
It replaces manual attendance tracking with a fast, secure, and digital solution.

---

## 🧠 Key Features

### 👨‍🏫 Admin / Teacher Features:
- Add, update, and delete student records
- Manage teachers and classes
- Mark daily attendance
- View attendance reports
- Dashboard for quick insights

### 👨‍🎓 Student Features:
- View attendance records
- Track performance
- Access results

---

## 🛠️ Tech Stack

- 🐍 Python
- 🌐 Django Framework
- 🗄️ SQLite Database
- 🎨 HTML, CSS

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/Darshit-Mehta-11/Student-Attandence-Management-System.git

# 2. Go to project folder
cd Student-Attandence-Management-System

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run migrations
python manage.py migrate

# 5. Start server
python manage.py runserver

👉 Open in browser:
http://127.0.0.1:8000/
